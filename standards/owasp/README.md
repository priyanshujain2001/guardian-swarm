# OWASP

This standard package captures OWASP Top 10 and related secure-by-design guidance for software reviews.

## Topics
- Injection
- Broken authentication
- Sensitive data exposure
- XML external entities
- Broken access control
- Security misconfiguration
- Cross-site scripting
- Insecure deserialization
- Vulnerable components
- Logging and monitoring failures

## Usage
Reference this standard from skills that perform security reviews or dependency triage.
